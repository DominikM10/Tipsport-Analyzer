This project is a data-driven analyzer that optimizes NHL fantasy lineups for Tipsport's daily fantasy game. Its core function is to identify high-value, under-priced players to generate "alpha" (excess returns) against the market's cash prices.

It moves beyond simple point averages by incorporating a statistically grounded approach, including dynamic performance weighting and special adjustments for high-potential rookies, to build a competitive and reproducible fantasy lineup.

- Rookie Amplification: Applies a statistical adjustment to promising rookies, identifying high-upside players before their price fully reflects their potential.

- Extensible Architecture: Designed with multiple modes (Greedy, Iterative, and ML-based) to allow for different levels of computational depth and analysis.
